# Employee Management System

A full-stack web application designed to manage employees and their leave requests efficiently. This project demonstrates real-world CRUD operations, REST API integration, and role-based leave management using modern web technologies.

---

## 🚀 Features

### 👨‍💼 Employee Management
- Add new employee records
- Update existing employee details
- Delete employee records
- View all employees
- Search employee by name or ID

### 📝 Leave Management
- Employees can apply for leave
- View leave request status
- Admin can view all leave requests
- Admin can approve or reject leave requests

---

## 🛠️ Tech Stack

### Frontend
- React.js

### Backend
- Spring Boot
- Spring Data JPA
- RESTful APIs

- MySQL

## ⚙️ Setup Instructions

### 🔧 Backend Setup
1. Open backend project in Eclipse or VS Code
2. Configure MySQL database:
   - Create database: `EmployeeManagementSystem`
3. Set environment variables:
   - DB_USERNAME
   - DB_PASSWORD
4. Run Spring Boot application
